# 🎮 Hangman Game

## 📌 Description
This is a simple Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship.

## ✨ Features
- Random word selection
- Letter guessing
- Limited attempts
- Win/Lose message
- Beginner-friendly Python project

## 🛠 Technologies Used
- Python 3

## ▶️ How to Run

1. Download the project.
2. Open the project folder.
3. Run the following command:

```bash
python codealpha_hangman_game.py
```
